myenv
=====

This project is for my current shell configurations. Just run `./install.sh`

**Plugins:**
- Vundle
- CtrlP
- Pathogen
- Systastic
- ShowFunc
- Greplace
- taglist
- tmuxline
- airline
- vim-fugitive
- vim-gutter
- snipmate
- vim-autoformat
- indentLine

Color scheme: jellybeans. You can also change it, adding it as a plugin
